---
title: Brew
description: Homebrew installs packages to their own directory and then symlinks their files into /opt/homebrew (on Apple Silicon).
date: 2023-12-21
---

**Install**

```sh
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**How to Install another dependencies**

- e.g. install visual-studio-code

```sh
$ brew install --cask visual-studio-code
```
